# Asset & Reference Inventory

Acquired 2026-07-06 per [cnc-clone-asset-sources.md](cnc-clone-asset-sources.md). Everything below is
gitignored — copyrighted game data, personal use only.

## data/assets/ — fully extracted game files (~2.7 GB, 11,911 files)

Produced by `tools/mix_extract.py`: all 112 MIX archives (including nested and
Blowfish-encrypted ones) recursively unpacked, 98.8% of files resolved to real names via XCC's
global mix database. The 139 leftovers are named `0x<hash>` with a sniffed extension. Layout
mirrors `data/extracted/`, one directory per MIX (e.g. `red_alert/allied/MAIN/conquer/`).

What's in there, by type:

| Type | Count | Contents |
|---|---|---|
| `.vqa` (2.0 GB) | 345 | FMV cutscenes |
| `.aud` (485 MB) | 1,637 | Music, SFX, EVA/unit voices |
| `.shp` | 2,171 | Unit/structure/animation sprites (need a `.pal` to render) |
| `.pal` | ~40 | 256-color palettes per theater (`temperat.pal` etc.) |
| `.tem`/`.sno`/`.des`/`.int` | ~4,400 | Terrain tile templates per theater |
| `.ini` | 721 | **Game rules + missions** — `red_alert/*/INSTALL/REDALERT/local/rules.ini` is the RA balance bible; `sc*.ini` are missions |
| `.wsa`/`.cps` | 604 | Fullscreen animations and images (menus, briefings) |

## data/ staging dirs (~9 GB total)

- `data/downloads/` — original freeware archives from files.cncnz.com (7 files, ~2.9 GB). Safe to
  delete once you're confident the extractions are good; re-downloadable from CNCNZ.
- `data/discs/` — the ISOs unpacked from those archives.
- `data/extracted/` — game files pulled off the ISOs. This is what the engine will read:

| Path | Contents |
|---|---|
| `tiberian_dawn/gdi/`, `nod/` | CONQUER.MIX (unit/structure sprites), GENERAL.MIX, DESERT/TEMPERAT/WINTER.MIX (terrain per theater), SOUNDS.MIX + AUD.MIX (SFX/voice), SCORES.MIX (music), MOVIES.MIX (FMV), INSTALL/CCLOCAL.MIX |
| `tiberian_dawn/covert_ops/` | Expansion missions + extra music, incl. INSTALL/SC-000/SC-001.MIX (missions), TRANSIT.MIX, SPEECH.MIX |
| `red_alert/allied/`, `soviet/` | MAIN.MIX (bulk of game data: movies/music/art), INSTALL/REDALERT.MIX (core game data: rules, sprites, local data) |
| `red_alert/counterstrike/`, `aftermath/` | Expansion MAIN.MIX archives |

The GDI and Nod discs (and Allied/Soviet) share most game data — they differ mainly in campaign
FMVs. For engine work, one disc per game is enough; the rest is there for completeness.

Remaining format work happens at load time in the engine: SHP/PAL → textures, AUD → PCM,
TMP terrain tiles, VQA playback (optional). OpenRA's readers in
`reference/OpenRA/OpenRA.Mods.Cnc/FileFormats/` are the working spec for all of these.

## tools/

- `mix_extract.py` — recursive MIX extractor (TD format + RA encrypted headers), ported from
  OpenRA's MixFile/BlowfishKeyProvider. Produced `data/assets/`. Idempotent (skips existing
  output dirs). Usage:
  `python mix_extract.py <extracted-root> <out-root> "xcc/global mix database.dat"`
- `blowfish_tables.py` — Blowfish constants, generated by `gen_blowfish_tables.py` from OpenRA's
  Blowfish.cs.
- `xcc/global mix database.dat` — XCC's hash→filename database (pulled out of the XCC installer).
- `XCC_Utilities.exe` — installer for XCC Mixer etc. (GUI alternative for browsing MIX contents).
- `7zip/` — portable 7-Zip 25.01 (`7z.exe`), used for zip/rar/iso.
- `extract-assets.ps1` — the script that produced `data/discs` + `data/extracted`. Idempotent
  (skips already-extracted targets).

## reference/ (shallow clones, GPL v3)

- `CnC_Tiberian_Dawn/`, `CnC_Red_Alert/` — original Westwood C++ source (reference only, won't
  build on modern toolchains).
- `CnC_Remastered_Collection/` — remaster source (C# + updated engine code).
- `OpenRA/` — modern open-source reimplementation; best working reference for MIX/SHP/PAL parsing
  and overall architecture.
